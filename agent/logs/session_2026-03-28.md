### 4. 添加 UserPromptSubmit Hook

**配置内容**:
```json
"UserPromptSubmit": [
  {
    "hooks": [
      {
        "type": "command",
        "command": "bash -c 'echo \"[ 自动附加上下文 ] 当前分支: $(git branch --show-current 2>/dev/null || echo unknown); 最近提交: $(git log --oneline -1 2>/dev/null || echo none)\"'",
        "timeout": 10
      }
    ]
  }
]
```

**效果**: 每次提交用户提示时自动显示分支和最近提交信息
